Q1. What are the new tags added in HTML5?

Ans. HTML5 introduced several new tags that were not present in previous versions of HTML. Some of the new tags added in HTML5 include:

1. header: Defines a header section for a document or a section.

2. footer:Defines a footer section for a document or a section.

3. nav:Defines a set of navigation links.

4. article: Represents an independent piece of content that can stand alone, such as a blog post or a news article.

5. section:Defines a section in a document, such as chapters, headers, footers, or any other thematic grouping.

6. aside:Represents content that is tangentially related to the content around it, such as sidebars or pull quotes.

7. main:Specifies the main content of a document.

8. figure:Represents self-contained content, such as images, diagrams, illustrations, code snippets, etc., along with an optional caption. < figcaption >

9. figcaption:Represents a caption or legend for a < figure > element.

10. video: Embeds a video into an HTML document.

11. audio:Embeds audio content into an HTML document.

12. canvas: Provides a container for graphics, animations, or other visual images using JavaScript.

13. progress:Represents the progress of a task or process, typically displayed as a progress bar.

14. meterRepresents a scalar measurement within a known range, such as disk usage or completion percentage.

15. time:Represents a specific period in time or a duration, with optional formatting and machine-readable data.

16. datalist: Provides a list of predefined options for input controls like < input > or < textarea >

17. details:Represents a disclosure widget from which the user can obtain additional information or controls.

18. summary:Represents a summary, caption, or legend for a < details > element.

19. dialog:Represents a dialog box or window.

Q2. How to embed audio and video in a webpage?

To embed audio and video in a webpage using HTML5, you can use the < audio > and < video > tags, respectively. Here's how you can do it:

1. Embedding Audio:

Use the < audio > tag to embed audio content in your webpage. Here's an example:

< audio controls > < source src="audio_file.mp3" type="audio/mpeg" > Your browser does not support the audio element. < /audio >

controls: Adds playback controls (play, pause, volume, etc.) to the audio player.

< source src="audio_file.mp3" type="audio/mpeg" >:

Specifies the audio file (audio_file.mp3) and its MIME type (audio/mpeg in this case). You can add multiple < source > elements to provide alternative audio formats for better compatibility across browsers.

Replace "audio_file.mp3" with the actual path to your audio file.

Embedding Video:Use the < video > tag to embed video content in your webpage. Here's an example:

< video controls width="640" height="360" >
< source src="video_file.mp4" type="video/mp4">
Your browser does not support the video element.
< /video >

controls:Adds playback controls (play, pause, volume, etc.) to the video player.

width="640" height="360":Specifies the width and height of the video player.

< source src="video_file.mp4" type="video/mp4" >:Specifies the video file (video_file.mp4) and its MIME type (video/mp4 in this case). Again, you can add multiple < source > elements for different video formats.

Replace "video_file.mp4" with the actual path to your video file.

Fallback Content:

The text "Your browser does not support the audio/video element." inside the < audio > and < video > tags is the fallback content. It will be displayed if the user's browser does not support the audio or video element.

Q4. Canvas and SVG tags

Ans. Both the < canvas > and < svg > tags are used for creating graphics and visual elements in HTML, but they have different approaches and purposes:

1. canvas Tag:

The < canvas > tag is used for drawing graphics, animations, and interactive content using JavaScript.

It provides a bitmap-based drawing surface that allows you to manipulate pixels directly through JavaScript.

The content drawn on a < canvas > is not part of the document's DOM structure, which means it cannot be accessed or manipulated using standard DOM methods.

Example of a canvas tag:

< canvas id="myCanvas" width="400" height="200" > </canvas>

In this example, id="myCanvas" provides an identifier for JavaScript to target this canvas element, and width="400" height="200" sets the dimensions of the canvas.

svg Tag:

The < svg > tag is used for creating scalable vector graphics (SVG) directly within HTML.

SVG graphics are vector-based, meaning they can be scaled without losing quality and are suitable for illustrations, diagrams, icons, and more.

SVG elements are part of the document's DOM structure, allowing them to be styled and manipulated using CSS and JavaScript.

Example of an svg tag:

< svg width="400" height="200"> < circle cx="100" cy="100" r="50" fill="red" / > < rect x="200" y="50" width="100" height="100" fill="blue" /> < /svg >
